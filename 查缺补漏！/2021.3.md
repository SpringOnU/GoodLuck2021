1. git分支合并

   `git checkout -b user` 新建分支

   `git add .` 

   `git commit -m "用户管理"` 

   `git push -u origin user` 推送

   `git branch` 查看分支

   `git checkout main `切换到main分支

   `git merge user` 合并user分支

   `git push `推送 此时文件都在云端 就不用再-u了

2. 创建新标签

   `git checkout -b goods_params`

   `git push -u origin goods_params`
